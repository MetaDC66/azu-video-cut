# 剪辑全程检查清单（每条视频开工打印一份，逐项勾选）

> 依据：references/workflow.md 四道确认门 + 历次经验沉淀。工具入口：`scripts/review_gen/generate.py`、`scripts/mix_finalize.py`。

## 素材核验
- [ ] 母版为用户初剪锁定版，规格 ffprobe 实测（分辨率/fps/时长）
- [ ] 所有截图/录屏素材 ffprobe + 目检，实际内容与文件名/口播一致

## 转录 + 名词确认
- [ ] 词级转录独立缓存（ElevenLabs/whisper）
- [ ] 疑似错词逐条列出让用户确认后替换

## 字幕
- [ ] 全部去标点（数字小数点保留，拉丁词间补空格）
- [ ] canon 校验：拼接字幕 == 权威口播稿逐字一致
- [ ] 时间线插入/延时改动时跨界条按实测 token 拆分，不按起点归类
- [ ] 高亮词表写入 subtitle-timeline.json 的 highlight（数据驱动）

## 覆盖层
- [ ] 槽位无重叠，scene.type 用注册钩子类型
- [ ] 关键帧静帧（每场景一张 1920×1080）用户过目确认后才渲染
- [ ] hyperframes check 0 error（瞬态对比度 warning 可接受）

## 抽检样片（G3 门，固定规则）
- [ ] 窗口覆盖一个场景交接点 + 高信息密度段
- [ ] 带完整混音（含长静音段窗口人声目标 -13.2，否则 -16）
- [ ] 实测 -14±0.3 LUFS / ≤-1.5 dBTP
- [ ] 用户通看确认后才进全片

## 完整审片版
- [ ] 响度参数按全片实测重算（禁止照抄抽检段）
- [ ] 渲染工程 master.mov inode 断言通过（强制重链，防陈旧硬链接）
- [ ] 抽 8–10 代表帧目检：字幕最顶层/无标点/90% 覆盖层/无闪屏/交接干净
- [ ] 音字同步证据链：包络互相关 lag≈0 + DTW token 偏差表每条 ≤0.15s

## 交付版
- [ ] 同工程 -q high 重渲染 + 同混音链，视频流 copy
- [ ] 记录 SHA-256；与审片版同刻帧比对（差 ≤2/255 为编码噪声）

## 封面
- [ ] 唯一承诺 → 发布标题与封面大字分开，用户逐字确认后才出图
- [ ] 角标 PIL 合成（左上 330px 留边 28px），不让生成模型画角标
- [ ] 270px 缩略图大字可读

## 发布前自查
- [ ] 违禁词/限流风险自查：调用 traffic-compass skill 检查发布标题 + 口播稿
- [ ] 确认发布标题与封面大字承诺一致

## Token 记账（每条视频两次）
- [ ] 开工时：在 `edit/<N>-master/token-log.md` 记一行「日期 / 阶段起点 / 当前累计 token 读数」
- [ ] 交付时：再记一行并算出本条总消耗，追加到 `复盘/发布记录.md` 对应行，用于新旧管线成本对照
