# 科技知识视频封面复用规范

## 使用边界

- 借鉴成熟科技内容封面的信息层级与视觉语法，不复制具体作品、人物形象、Logo、构图或专有装饰。
- 每次先确认发布标题与封面大字，再生成封面；未经确认不改字、不出图。
- 发布标题负责完整表达，封面大字负责移动端第一眼识别，两者不必逐字相同，但必须承诺一致。

## 已确认视觉基线

- 比例：3:4 竖版；当前基准成品为 1086×1448 PNG。
- 主体：使用用户本人的成片截图，人物占画面约 40%–55%，优先放在右侧或右下。
- 个人识别点：保留人物身份特征（标志性配饰/发型/服装）和真实手势，不擅自移除或重塑身份。
- 背景：蓝黑暗色科技空间，以发光代码面板、流程、数据或能量流承担“证据/机制”功能。
- 光色：蓝青为主，橙红作为竞争或冲突色；保持明亮、现代，避免惊悚、末日和霓虹过载。
- 大字：2–3 行，每行尽量 4–7 个汉字；粗重窄体无衬线，深色描边和轻阴影，缩略图尺寸仍能读清。
- 颜色层级：第一行蓝青或蓝橙；第二行白色；结论或问句使用最大号黄白色。
- 画面纪律：不增加平台 UI、水印、第三方 Logo、细小副文案或无法核验的截图。
- 品牌角标例外：封面统一带「<频道名>」个人标签（首次配置见 setup.md 第三步），AI 出图后 PIL 后期合成（不让生成模型画角标），默认左上角、宽 330px、留边 28px。

## 选图规则

1. 优先选择正对镜头、口型自然、手势明确的帧（全片每 2s 抽帧 + 拉普拉斯方差锐度排名 + 安静窗口 0.2s 细扫，避开运动模糊）。
2. 人物面部与身份特征必须清楚，文字不得压住主体。
3. 手势用于推动标题含义，不为“有动作”选择畸变或遮脸帧。
4. 背景可以重构，但人物身份特征和主要服装应尽量保持。

## 固定工作流

1. 读取成片字幕，提炼正文能够兑现的唯一承诺。
2. 分开给出“发布标题”和“封面三行大字”。
3. 用户逐字确认标题。
4. 从成片抽取 6–8 个候选人物帧，选择手势与表情最匹配的一帧。
5. 生成一版 3:4 封面并核对：文字、人物、手指、身份特征、Logo、水印和缩略图可读性。
6. 生成图若带「AI生成」小字，2:3→3:4 上下裁切（约 85/86 px）可同时去除；缩放至 1086×1448。
7. 保存到封面输出目录，使用 `第N条视频封面-v1.png` 这类版本化命名，不覆盖旧稿。

## 可复用提示词骨架

```text
用途：3:4 竖版科技知识视频封面。
人物参考：保留用户本人的身份特征（标志性配饰/发型/服装）和真实手势；人物置于右侧，使用中近景。
风格参考：只借鉴暗色科技感、单一大主体和粗体三行大字的信息层级，不复制任何具体创作者封面。
背景：蓝黑科技工作区，加入与选题机制相关的发光数据、代码、流程或竞争光流，不使用虚假证据。
文字（逐字）：
“[第一行]”
“[第二行]”
“[第三行]”
排版：第一行蓝青/橙色，第二行白色，第三行最大号黄白色；粗重窄体、深色描边、移动端可读。
禁止：额外文字、Logo、水印、角标、平台 UI、错别字、人物身份漂移、身份特征改变、畸形手指、恐怖氛围、视觉拥挤。
```
